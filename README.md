# DuReader

DuReader is a manga reader for [LANraragi](https://github.com/Difegue/LANraragi)

## Control

- Next page: tap
- Navigation menu: long tap
- Previous page: not supported yet

## Test on your device

- Clone this project
- Open it in Xcode
- Follow the [guide](https://developer.apple.com/documentation/xcode/running_your_app_in_the_simulator_or_on_a_device)
